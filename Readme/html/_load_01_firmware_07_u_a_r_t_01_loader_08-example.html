Load Firmware(UART Loader)

This document demonstrates how to use the STM32 Flash Loader to download firmware.

Software Tool Download

http://www.st.com/content/st_com/en/products/development-tools/software-development-tools/stm32-software-development-tools/stm32-programmers/flasher-stm32.html

Hardware

Pull Boot0 to high when power on, then module will enter IAP mode.
Connect device's micro usb port to your PC.

Note
Use pin NRST if MCU’s in low power mode to wake the MCU up.

Upgrade Step

Step 1

Pull Boot0 to high when power on, MCU enter IAP mode.

Step 2

Open Flash Loader Demonstrator and select COM port.

Step 3

Press “Next”, check if MCU enter IAP mode. If it is ok, show below screen.

Warning
If it is fail, show below screen. Please reset module.

Step 4

Select module size “STM32L0_x3_x2_x1_192k”.

Step 5

Select hex file.

Step 6

Successfully download will show as below.

Step 7

Remove boot0 pin, and reset module.
